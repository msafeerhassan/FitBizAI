# FitBizAI

## A Personal Health, Fitness and Productivity Tracking Web App powered by AI

### Python-Streamlit based

#### Features:
1. Daily Logging of Diet, Water, Workout, Work and Additional Context
2. AI-Generated Daily Coach Report with Scores, Positive Points, Negative Points and next day's Goals
3. AI Analysis of fulfillment of previous goals
4. Personal Daily Water, Workout and Work Target Setting Page
5. Goal VS Target Tracking on Dashboard
6. Mandatory Fortnightly Report with weight and facial photos input for AI Analysis
7. Photo's Gallery for Facial Pictures logged in Fortnightly Reports
8. Insights Page with detailed and accurate line graphs
9. Daily Report Streak and Goal Streak Tracking
10. Achievements Page with Badges
11. Weekly AI-Generated Reports.
12. Page to chat with your AI Coach
13. Page to manage your logs
14. Local JSON-Based Storage for easy setup.

#### How to test it online:
1. Go over to [FitBizAI Demo](https://fitbizai.streamlit.app/)
2. Test it :)

#### How to use it locally:
1. Clone the repository and open the project folder.
2. Install required dependencies (Streamlit, Pandas, OpenRouter, python-dotenv)
3. Create a `.env` file in root folder and add your Hack Club AI API Key. You can get one at: [Hack Club AI](https://ai.hackclub.com/).
4. Run the app using `streamlit run app.py`.

**NOTE: Since some features are only unlocked when extensive data is logged, I have added dummy `userProfile.json`, `record.json`, `chatHistory.py` files so you may test it without logging all data.**
